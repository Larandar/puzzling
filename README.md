# Larandar's code puzzle solving

## How to use

### Initialization

Create a `puzzling.toml` file at the root of the crate

```toml
[advent_of_code]
session_secret = "YOUR_ADVENT_OF_CODE_SESSION_ID_HERE"
```

### Daily runner

```bash
./advent.fish YYYY DD
```
